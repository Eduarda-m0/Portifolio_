# Quem Sou Eu

Projeto de portfólio pessoal em HTML, CSS e JavaScript.

## Sobre

Este site apresenta um perfil pessoal e profissional. A página foi criada como um portfólio simples para mostrar:

- informações pessoais e história;
- pontos fortes e características;
- formação e cursos concluídos;
- visão profissional;
- um projeto de TCC (Healthy Company);
- links para redes sociais.

## Estrutura do Projeto

- `index.html` - página principal do site.
- `css/style.css` - estilos visuais e layout.
- `js/script.js` - comportamento interativo do menu, header e carrossel.
- `img/` - imagens usadas no site.

## Funcionalidades

- menu responsivo para navegação interna;
- header que troca de estilo ao rolar a página;
- animação no título da aba do navegador;
- carrossel de imagens no projeto;
- seções de apresentação, história, formação, visão e contato.

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript puro
- Font Awesome (ícones)
- Google Fonts (`Inter`)

## Como usar

1. Abra o arquivo `index.html` em um navegador.
2. Navegue pelas seções usando o menu ou rolando a página.
3. Veja o carrossel de imagens na seção "Meu Projeto (TCC)".

## Observações

- O site é estático e não requer servidor.
- É recomendado usar um navegador moderno para melhor visualização.
